## 機能

- タスクの追加・編集・削除
- タスクの完了管理
- カテゴリ管理
- カテゴリカラー設定
- タスク詳細画面
- 月別達成率
- 今日・過去7日・過去30日・総達成率
- 連続達成日数
- 月切り替え機能
- レスポンシブを意識したUI
- LocalStorageによるデータ保存

---

## 使用技術

- React
- TypeScript
- React Router
- Context API
- CSS
- LocalStorage

---

## 起動方法

```bash
git clone https://github.com/your-account/todo-list-react-ts.git

cd todo-list-react-ts

npm install

npm run dev
```
